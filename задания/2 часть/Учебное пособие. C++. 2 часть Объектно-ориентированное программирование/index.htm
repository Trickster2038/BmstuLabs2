Объектно-ориентированное программирование на языке C++ в среде Microsoft Visual Studio  2008. Электронное учебное издание. -  М.: МГТУ имени Н.Э. Баумана, 2013. 174 с.

Представлен теоретический материал, содержащий описание средств, механизмов и методов разработки программ на языке программирования С++ в среде Microsoft Visual Studio 2008 с использованием объектной технологии, изучение которой предусмотрено программой дисциплины «Объектно-ориентированное программирование» кафедры «Компьютерные системы и сети» МГТУ им. Н.Э. Баумана. Изложение сопровождается поясняющими рисунками, диаграммами на языке UML и примерами программ, предлагаются вопросы и задачи для самоконтроля. Рассмотрены приемы, существенно сокращающие размер кода программы, а также средства помехозащитного программирования.

Для студентов МГТУ имени Н.Э. Баумана, обучающихся по программам бакалавриата направления «Информатика и вычислительная техника».
